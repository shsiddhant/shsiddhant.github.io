Hi, I’m Siddhant

I build data pipelines and analytical systems, mostly in Python and SQL. My work usually revolves around data modeling, orchestration, and figuring out how to move from raw data to ready-for-analytics.

A lot of my projects come from things I already care about, like cricket and music, which is why they tend to focus on tracking, patterns, and long-term analysis.

This blog is where I write about the process behind those systems, including but not limited to:

- Analytical methods and ideas with the mathematics they involve.
- Problem solving with curiosity as the backbone.
- Cricket and Music 

The blog it built using [Jekyll](https://jekyllrb.com/) with the [Reverie](https://jekyllthemes.io/theme/reverie) theme and hosted on Github Pages at https://shsiddhant.github.io/.
